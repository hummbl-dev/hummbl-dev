# Social Assets Evidence Posture

This folder contains **social derivatives** of the HUMMBL research corpus.

- These assets are optimized for compression, clarity, and memorability.
- They are not optimized for full nuance.

Social copy should be treated as the **highest-drift derivative layer** in this repo:

- percentages and benchmark claims can go stale
- legal and regulatory summaries can become over-absolute
- marketing phrasing can outrun the underlying evidence

Before external reuse beyond social posting, re-verify the underlying research and use the repo-level posture note:

- [`../../../PROOF_AND_RESEARCH_EVIDENCE_POSTURE.md`](../../../PROOF_AND_RESEARCH_EVIDENCE_POSTURE.md)
