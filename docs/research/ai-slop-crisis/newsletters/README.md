# Newsletter Assets Evidence Posture

This folder contains **newsletter drafts and publication variants** derived from the research corpus.

- Newsletters compress complex research into short, high-signal narratives.
- That makes them useful communication tools, but also a high-risk place for overstatement, drift, and outdated numbers.

Treat newsletter copy as **research-backed editorial synthesis**, not as canonical source material.

Before reusing claims externally:

- verify dates, percentages, and legal/regulatory timing
- recheck case-law posture and insurance-market claims
- avoid treating urgency language as direct evidence
